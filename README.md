# Lumina Dashboard (Dev)

Projeto de desenvolvimento do dashboard local para Hubitat.

## Estrutura

- `/dist`: Versões compiladas (Single File HTML) prontas para uso no Hubitat File Manager.
- `/hubitat-drivers`: Códigos Groovy (Drivers e Apps) necessários no Hubitat.
- `/src`: (Futuro) Código fonte React original.

## Instalação

1. Suba o arquivo de `/dist/index.html` para o File Manager do Hubitat.
2. Instale os drivers de `/hubitat-drivers` na seção "Drivers Code".
